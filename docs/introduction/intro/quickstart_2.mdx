### Introduction to Entities

- Each unique user is represented by a unique entity id.
- If you have two users, Jessica and Melissa, they will have unique entity ids.

### Introduction to Connected Accounts

- When a user connects their account, a connected_account object is created.
- This object securely stores authentication data such as access tokens and refresh tokens.
- Turn natural language into Gmail actions in minutes. Follow these 5 steps to connect your user to gmail and send an email:

<Info>
    **What you'll achieve:** By the end of this guide, you'll connect your user to their Gmail and Send an Email through Natural Language.
</Info>

<Steps>

    <Step title="Install Composio">
        Choose your preferred language:

        <Tabs>
            <Tab title="Python">
                <CodeGroup>
                    ```bash Install Packages
                    pip install composio_core composio_openai
                    ```
                </CodeGroup>
            </Tab>
            <Tab title="JavaScript">
                <CodeGroup>
                    ```bash Install Packages
                    npm install composio-core openai
                    pip install composio_core  # for CLI (optional)
                    ```
                </CodeGroup>
                <Note>
                    CLI binary package coming soon. Skip `pip install` if not using CLI.
                </Note>
            </Tab>
        </Tabs>
    </Step>

    <Step title="Authenticate Gmail Account">
        We'll use **`Jessica`** as our example user. Choose your method:

        <Tabs>
            <Tab title="CLI (Quickest)">
                <CodeGroup>
                    ```bash Authenticate Jessica's Gmail Account
                    composio add gmail -e "Jessica" # Launches Gmail login
                    ```
                </CodeGroup>
            </Tab>
            <Tab title="Python">
                <CodeGroup>
                    ```python Authenticate Jessica's Gmail Account
                    from composio import ComposioToolSet, App

                    toolset = ComposioToolSet(entity_id="Jessica")
                    entity = toolset.get_entity()
                    request = entity.initiate_connection(App.GMAIL)

                    print(f"Open this URL to authenticate: {request.redirectUrl}")
                    ```
                </CodeGroup>
            </Tab>
            <Tab title="JavaScript">
                <CodeGroup>
                    ```javascript Authenticate Jessica's Gmail Account
                    import { Composio } from "composio-core";

                    const client = new Composio(
                    process.env.COMPOSIO_API_KEY
                    );

                    const entity = await client.getEntity("Jessica");
                    const connection = await entity.initiateConnection('gmail');

                    console.log(`Open this URL to authenticate: ${connection.redirectUrl}`);
                    ```
                </CodeGroup>
            </Tab>
        </Tabs>

        <Warning>
            Ensure you complete the authentication process by following the URL provided in your console.
        </Warning>

    </Step>

    <Step title="Initialize Composio and OpenAI">
        Set up your development environment:

        <Tabs>
            <Tab title="Python">
                <CodeGroup>
                    ```python Initialize Clients
                    from composio_openai import ComposioToolSet, Action
                    from openai import OpenAI

                    openai_client = OpenAI()
                    composio_toolset = ComposioToolSet(entity_id="Jessica")
                    ```
                </CodeGroup>
            </Tab>
            <Tab title="JavaScript">
                <CodeGroup>
                    ```javascript Initialize Clients
                    import { OpenAI } from "openai";
                    import { OpenAIToolSet } from "composio-core";

                    const openai_client = new OpenAI({
                    apiKey: process.env.OPENAI_API_KEY
                });

                    const composio_toolset = new OpenAIToolSet({
                    apiKey: process.env.COMPOSIO_API_KEY,
                    entityId: "Jessica",
                });
                    ```
                </CodeGroup>
            </Tab>
        </Tabs>

        <Tip>
            Don't forget to set your `COMPOSIO_API_KEY` and `OPENAI_API_KEY` in your environment variables.
        </Tip>

    </Step>

    <Step title="Fetch Gmail Actions & Interact with LLM">
        Now, let's star a repository using natural language:

        <Tabs>
            <Tab title="Python">
                <CodeGroup>
                    ```python Star a repo using LLM
                    tools = composio_toolset.get_actions(actions=[Action.GMAIL_SEND_EMAIL])

                    task = "Send an email to [ENTER ANY EMAIL ID] saying hi"

                    response = openai_client.chat.completions.create(
                    model="gpt-4-turbo-preview",
                    tools=tools,
                    messages=
                        [
                            {"role": "system", "content": "You are a helpful assistant."},
                            {"role": "user", "content": task},
                        ],
                    )
                    ```
                </CodeGroup>
            </Tab>
            <Tab title="JavaScript">
                <CodeGroup>
                    ```javascript Star a repo using LLM
                    const tools = await composio_toolset.getActions({
                    actions: ["gmail_send_email"]
                });

                    const instruction = "Send an email to [ENTER ANY EMAIL ID] saying hi";

                    const response = await openai_client.chat.completions.create({
                    model: "gpt-4-turbo-preview",
                    messages: [{ role: "user", content: instruction }],
                    tools: tools,
                    tool_choice: "auto",
                });
                    ```
                </CodeGroup>
            </Tab>
            <Tab title="API">
            You can get a list of Actions for a particular App like this. In this case we obtain all the Github Actions
            <CodeGroup>
              ```curl curl
              curl --request GET \
              --url 'https://backend.composio.dev/api/v2/actions?appNames=GMAIL' \
              --header 'X-API-Key: ADD YOUR API KEY HERE'
              ```
            </CodeGroup>
            </Tab>
        </Tabs>
    </Step>

    <Step title="Execute Tool Calls">
        Finally, let's execute the action determined by the LLM:

        <Tabs>
            <Tab title="Python">
                <CodeGroup>
                    ```python Execute Function calls for Jessica
                    result = composio_toolset.handle_tool_calls(response)
                    print(result)
                    ```
                </CodeGroup>
            </Tab>
            <Tab title="JavaScript">
                <CodeGroup>
                    ```javascript Execute Function calls for Jessica
                    const result = await composio_toolset.handleToolCall(response);
                    console.log(result);
                    ```
                </CodeGroup>
            </Tab>

            <Tab title="API">
            <CodeGroup>
            ```curl curl
              curl --request POST \
              --url https://backend.composio.dev/api/v2/actions/GMAIL_SEND_EMAIL/execute \
              --header 'Content-Type: application/json' \
              --header 'X-API-Key: ADD YOUR API KEY HERE' \
              --data '{
              "entityId": "Jessica",
              "input": {
                  "name": "prathit3.14@gmail.com",
                  "content": "Hi!"
              }
              }'
             ```
            </CodeGroup>
            </Tab>

        </Tabs>
    </Step>
